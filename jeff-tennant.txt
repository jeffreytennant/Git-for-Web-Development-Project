    37. Walk me through Lambda's git flow. 
    1. Fork the project
    2. Be sure your name is to the left of the project
    3. Clone the repo by clicking "code" and copying the HTTPS URL
    4. In your terminal, use cd and go to the directory you're wanting to use for the repo
    5. Type git clone *paste link from git hub* and press enter
    6. cd into the name of the repo just cloned
    7. Type git checkout -b 'firstname-lastname'
    8. Type code . to launch visual code studio
    9. Push the changes to github by typing git push -u origin firstname-lastname    
    10.Submit pull request in github
    11.Go to forked project and under branch, choose your name 
    12.Click new pull request
    13.Click on the dropddown button that says base repository and change it to be your forked repo
    14.Click create pull request
